# FastSpring Checkout Components Examples

This repository contains documentation and example integrations demonstrating
how to use FastSpring Checkout Components.

## Overview

Each example is a self-contained project showing a different integration 
approach. Browse the `examples/` directory to find a starting point that matches
your stack.

## Examples

- [`examples/vanilla/basic`](examples/vanilla/basic) — A minimal vanilla JavaScript example.

## Getting Started

1. Clone this repository.
2. Navigate to the example you'd like to try.
3. Follow the instructions in that example's `README.md`.
